YouTube Authentication

      

        YouTube OAuth2 was removed from yt-dlp. Cookie-based auth is the only supported method.
        For best results exporting YouTube cookies:
      

      
    
        
  1. Open a fresh Chrome/Edge window in a normal (non-incognito) tab
    2. 
        
  2. Sign in to YouTube and watch any video to refresh session cookies
    3. 
        
  3. Use the Get cookies.txt LOCALLY extension and export for youtube.com
    4. 
        
  4. Upload the file using the cookie manager below
    5. 
      

      

        YouTube cookies typically last 1–6 months. Re-export if you start seeing sign-in errors in job logs.
      

    


    
    

      
Browser Cookies
